테이블스페이스 용량 확인 쿼리

2018-09-09
DBMS, ORACLE, TABLESPACE

업무를 하다보면 계정의 테이블스페이스의 현재 상태를 체크해야할 일이 생깁니다.

예를 들어 데이터를 이관해야하는 경우, AUTOEXTENSIBLE가 YES일지라도 테이블스페이스의 현재 상태는 체크해주는게 좋습니다.

해당 쿼리 실행 시 총 용량과 현재 사용량, 잔여용량, 사용률 등을 확인할 수 있습니다.
